# Invoice Application

This is a simple Java Swing application for creating and displaying invoices.

## Description

The application prompts the user to enter invoice details, calculates the total with a discount if applicable, and displays the invoice information in a dialog box. The user can create multiple invoices until they choose to exit.

## How to Run

1.  **Compile the Java files:**
    Open a terminal or command prompt in the project directory and run the following command to compile the source code:
    ```bash
    javac InvoiceApp.java Invoice.java
    ```

2.  **Run the application:**
    After successful compilation, run the application with this command:
    ```bash
    java InvoiceApp
    ```

## Usage

1.  When the application starts, a dialog box will appear asking for the invoice number.
2.  Enter the invoice number and click "OK".
3.  Another dialog box will appear asking for the invoice total.
4.  Enter the total and click "OK".
5.  A third dialog box will appear asking for the invoice type.
    *   Enter `1` for a regular invoice.
    *   Enter `2` for a discounted invoice (10% discount).
6.  Click "OK".
7.  An invoice summary will be displayed.
8.  To create another invoice, press "Enter" or click "OK".
9.  To exit the application, enter `x` and click "OK".
